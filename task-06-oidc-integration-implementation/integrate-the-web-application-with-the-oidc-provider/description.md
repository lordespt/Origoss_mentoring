# Subtask: Integrate The Web Application With The Oidc Provider

## Purpose
Secure the Task 1 web application.

## Task Details & Parameters
Integrate the Task 1 frontend or an authenticating proxy (like OAuth2-Proxy) to require login via the deployed OIDC provider before accessing the application.

## Acceptance Criteria
- Users are redirected to the OIDC login page.
- Successful authentication grants access to the Task 1 application.
