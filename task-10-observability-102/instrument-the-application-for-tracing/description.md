# Subtask: Instrument The Application For Tracing

## Purpose
Trace requests through the Task 1 application.

## Task Details & Parameters
Deploy the chosen tracing backend. Instrument the Task 1 frontend and backend (using OpenTelemetry auto-instrumentation or SDKs) to send traces.

## Acceptance Criteria
- Complete traces showing requests originating from the frontend and reaching the backend/database are visible in the tracing UI.
